# Code source

Use the following basis to perform this exercise

1) Clone this repository and execution the commands

```bash
$ composer install
```

2) Run this project

```bash
$ docker-compose up -d
$ composer install -n
```

3) It's up to you to code!

Try PHPMyAdmin => [http://127.0.0.1:8085](http://127.0.0.1:8085)
